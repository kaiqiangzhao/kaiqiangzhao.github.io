python编码

编码

1 编码历史

为了处理英文字符，产生了ASCII码

为了处理中文字符，产生了GB2312

为了处理各国字符，产生了Unicode

为了提高Unicode存储和传输性能，产生UTF-8，它是Unicode的一种实现

2 python2

python2中的默认的字符编码为ASCII码

python2中字符串有str和unicode两种类型，str有各种编码的区别，Unicode是编码的标准形式

python2中可以直接查看到unicode的字符串

3 decode() 和 encode()

decode() 方法将其他编码字符串转化为Unicode编码字符

encode() 方法将unicode编码字符串转换为其它编码字符

Python2 的对于字符编码额转换要以unicode作为”中间人“进行转换。

python3

Python 3的源码.py文件 的默认编码方式为UTF-8，所以在Python3中你可以不用在py脚本中写coding声明，并且系统传递给python的字符不再受系统默认编码的影响，统一为unicode编码。

将字符串和字节序列做了区别，字符串str是字符串标准形式与2.x中unicode类似，bytes类似2.x中的str有各种编码区别。bytes通过解码转化成str,str通过编码转化成bytes。

我们看到当一个中文字符出现在一个list(或tuple、dict)中时，它并不会被显示为一个中文而是字节串。但当该字符串从list中提取出来再print时就能够正常显示为中文。字节串是所有字符在python中的“本质”形态，所以你可以简单的理解为list中呈现出的字节串是给计算机看的。
